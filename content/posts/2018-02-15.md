获取所有包信息：

```
pkgutil --pkgs | grep com.cisco
```

<!--more-->

删除所有遗留包：

```
sudo find / -name "*cisco*" | xargs rm -rf
sudo rm -rf /private/var/db/receipts/com.cisco.pkg.anyconnect.*
```
